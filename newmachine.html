Configure development environment¶

  • Configure Terminal

  • Install Homebrew

    $ ruby -e “$(curl -fsSL https://gist.github.com/raw/323731/install_homebrew.rb)”

  • Install Git

    $ brew install git

  • Install Hub

    $ brew install hub

  • Install Git-Flow

    $ brew install git-flow

  • Install my dotfiles

    $ cd ~ $ git clone http://github.com/bkonkle/dotfiles.git .dotfiles $ cd .dotfiles $ echo myreallylongapikeykeyfromgit > gitapikey $ rake install $ . ~/.bashrc

  • Install Python and Pip

    $ brew install python pip

  • Clone TextMate themes

    $ git clone bkonkle/TextMate-Themes ~/Library/ApplicationSupport/TextMate/Themes

Install TextMate plugins¶

  • [MissingSidebar](http://github.com/jezdez/textmate-missingdrawer/)

  • [AckMate](http://github.com/protocool/AckMate)

  • GetBundles

    $ mkdir -p ~/Library/ApplicationSupport/TextMate/Bundles $ cd !$ $ svn co http://svn.textmate.org/trunk/Review/Bundles/GetBundles.tmbundle/ $ osascript -e ‘tell app “TextMate” to reload bundles’

  • Install Python Django (by adamv)

  • Install Django Templates (by adamv)

Finishing up¶

  • Run Software Update again
